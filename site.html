Linguagem C++

C++ é uma linguagem independente do C, mas que surgiu a partir da linguagem C. O intuito era permitir trabalhar mais facilmente com orientação a objetos. Foi criado por Bjarne Stroustrup, e apareceu pela primeira vez em 1979.

Atualmente, é muito utilizado para programar:

- aplicativos desktop (como o Photoshop, o Google Chrome, o Firefox, o Microsoft Office)

- sistemas operacionais (por exemplo, parte do Windows)

- motores de jogos (por exemplo, o RAGE da Rockstar)

- bancos de dados (como partes do MySQL)

- firmware

- drivers

Linguagem C

A linguagem C surgiu em 1972 nos Bell Labs, a partir da necessidade de programar utilitários para o sistema operacional Unix, e também para reescrever o Unix. Ela foi criada por Dennis Ritchie a partir da linguagem B.

A linguagem C continua sendo muito utilizada, principalmente para programar:

- compiladores (por exemplo, o gcc)

- sitemas operacionais (por exemplo, o kernel do Linux e grande parte do Windows)

- interpretadores de outras linguagens (por exemplo, os interpretadores do Java (JVM), do Python (CPython) e do PHP (Zend))

- bancos de dados (por exemplo, o PostgreSQL e a maior parte do MySQL)

- muitos aplicativos de software livre (como o GIT)

- compiladores (por exemplo, o gcc)

- drivers de dispositivos de hardware

- firmware

Assembly

O assembly é uma representação (mais legível a humanos) dos comandos de máquina que o processador executa. O assembly não é uma única linguagem. De modo geral, cada arquitetura de processador possui a sua linguagem assembly. E não necessariamente todos os processadores de uma mesma arquitetura têm todos os comandos assembly de sua arquitetura (mas sempre possuem os principais).

A maioria dos computadores do tipo desktop e laptop hoje em dia trabalham com a arquitetura x86-64. Enquanto a maioria de celulares e tablets rodam a arquitetura ARM.

Atualmente, o assembly é utilizado principalmente para programar:

- firmware

- drivers

- compiladores

- sistemas operacionais (como pequenas partes do Windows e do Linux)
